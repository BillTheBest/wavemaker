Class Detail
wm.LiveForm()
LiveForm displays a set of editors for a data type. LiveForm's editors display the data in its dataSet property. Output data is provided via the dataOutput property. LiveForm can directly perform operations on the dataOuput without the need for additional services, it generates a set of editors automatically. An additional editPanel widget is added by default so that operations are available without additional user setup.
